Detailed Description

Adaptateur qui transforme les structures brutes (Module 1) en objets C++ typés (Module 2) prêts pour l'analyse de traçabilité.

AJOUT : les préfixes SSS/SRS/SDD acceptent désormais plusieurs valeurs séparées par des virgules (ex: "CLIENT, A, SSS"). Le préfixe global accepte aussi plusieurs valeurs (ex: "SPEC_, EX_, REQ_").
