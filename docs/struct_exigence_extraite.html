Detailed Description

Structure représentant une exigence extraite d'un fichier source.

Champs de base : identifiant : identifiant extrait par la regex (ex: EXIGENCE_CLIENT_A_F01) contenuBrut : texte descriptif de l'exigence fichierSource : chemin du fichier d'origine formatSource : extension détectée (csv, xls, docx) numeroLigne : numéro de ligne dans le fichier source

Métadonnées de traçabilité : tracabilite : IDs SSS couverts par cette SRS (colonne 2 CSV / ligne "Traçabilité :" docx) cibles : cibles logicielles/matérielles (ligne "Cible :" docx) necessaireA : projets/clients concernés (ligne "Nécessaire à :" docx) alloueA : projets auxquels la SSS est allouée (ligne "Alloué à :" docx) developpe : true si "Développé" figure dans le bloc docx
