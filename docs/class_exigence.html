Detailed Description

Classe mère abstraite représentant une exigence générique.

Contient les attributs et comportements communs à tous les types d'exigences (SSS, SRS, SDD) : identifiant, contenu, type et statut. Le destructeur est virtuel pour garantir une destruction correcte via un pointeur de classe mère.
